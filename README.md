# Training Tracker

Eine kleine lokale Web-App für das iPhone.

## Funktionen

- Training erfassen, auch mehrere Trainings auf einmal
- Automatische Berechnung: 2 gegebene Trainings = 1 kostenloses Privattraining
- Anzeige auf einen Blick: offene Guthaben, Trainings gegeben, verdient, eingelöst
- Privattraining als eingelöst markieren
- Verlauf für Trainings und Einlösungen
- Backup exportieren/importieren
- Daten werden lokal im Browser gespeichert

## Nutzung auf dem iPhone

### Einfach lokal öffnen

1. ZIP-Datei entpacken.
2. `index.html` auf dem iPhone öffnen, zum Beispiel aus der Dateien-App.
3. Die App speichert lokal im Browser.

### Als Home-Screen-App nutzen

Für eine echte Home-Screen-Web-App muss der Ordner über HTTPS geöffnet werden, z.B. über GitHub Pages, Netlify oder einen kleinen privaten Webserver. Danach in Safari öffnen und `Teilen` → `Zum Home-Bildschirm` wählen.

Wichtig: Die gespeicherten Daten hängen am Ort/URL, wo die App geöffnet wird. Wenn du die App später unter einer anderen URL öffnest, ist der Speicher separat. Darum regelmässig `Backup exportieren` nutzen.
